### Steps To Follow
	
	> Creating Virtual Environement First
		
		For Windows:
		
		`	
			pip install virtualenv
			python venv -m venv
		`
		
		For Ubuntu:
		
		`
			pip install virtualenv
			python3 venv -m venv
		` 
		or 
		
		`
			python venv -m venv
		`
	
	> Activating Environment

		For Ubuntu User:
			> source venv/bin/activate
	
		For Windows User:
			.\venv\Scripts\activate

	> Installing the Requirements 
	
		`
			pip install - r requirements.txt
		`
	
	> Run The Project
		`
			python3 app.y
		`
